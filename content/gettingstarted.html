Getting Started

You will need to setup your development environment to use this solution. This guide assumes you are familiar with using Visual Studio 2015 and you will need above the Basic version to execute the unit test project. Development requirements:
  • Visual Studio 2015 http://visualstudio.com
  • Azure SDK http://azure.microsoft.com/en-us/downloads/
  • Azure Storage Emulator or Azure Subscription with Storage account and access key http://azure.microsoft.com/en-us/documentation/articles/storage-dotnet-how-to-use-tables/

Identity 2.0

ElCamino.AspNet.Identity.AzureTable (ASP.NET Identity 2.0 framework) NuGet


Use this package to target the .NET Framework 4.5 and Microsoft.AspNet.Identity.Core 2.2 for Mvc 5. Included in the source is a MVC 5 Website project (targeting .NET Framework 4.5) that is an example of removing the EntityFramework and Identity EntityFramework references and replacing them with the Azure Table storage provider. The source of the sample project can be found Source Code /sample/samplemvc and the walk-through can be found here Sample Mvc Walkthrough .

Identity Core 1.0

ElCamino.AspNetCore.Identity.AzureTable (ASP.NET Identity Core 1.0 framework) NuGet


Use this package to target the .NET Framework 4.51 or .NET Core 1.0 and Microsoft.AspNetCore.Identity 1.0 for Mvc 6. Included in the source is a MVC 6 Website project (targeting .NET Core 1.0) that is an example of removing the EntityFramework and Identity EntityFramework references and replacing them with the Azure Table storage provider.
